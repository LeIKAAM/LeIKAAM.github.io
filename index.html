Installation complète avec DLCs :

  1. Installe Steam et connecte-toi.
  2. Dans Steam, coche Outils dans ta bibliothèque.
  3. Installe Call of Duty: Modern Warfare 3 Dedicated Server.
  4. Voir l'image d'aide pour trouver l'outil.
  5. Télécharge les DLCs via le lien Mega.
  6. Sur Steam : Clic droit sur MW3 (Server) > Gérer > Parcourir fichiers locaux.
  7. Glisse le contenu des DLCs dans ce dossier.
  8. Crée un compte Plutonium.
  9. Lance Plutonium, sélectionne MW3, clique Setup et choisis le dossier.
