Tutoriel Complet
Tuto pour hack la Switch

1. Matériel Requis

Carte SD : 64Go minimum recommandé.
64Go | 128Go | 256Go

Accessoires :
- Câble USB-C pour relier au PC : Lien
- RCM Jig (indispensable) : Lien

2. Vérifier si votre Switch est compatible

Seules les Switch de "première génération" (V1 non patchées) sont hackables facilement. Vérifiez votre numéro de série (sous la console).

Checker mon Numéro de Série

3. Entrer en mode RCM

1. Éteignez complètement la Switch.
2. Insérez le RCM Jig dans le rail du Joy-Con droit.
3. Maintenez Volume + et appuyez sur Power.
4. L'écran doit rester noir. S'il s'allume, recommencez.

4. Préparer la Carte SD

ATTENTION : Sauvegardez votre dossier "Nintendo" sur PC avant de commencer !

1. Téléchargez le Pack Switch (voir onglet au-dessus).
2. Copiez tout le contenu à la racine de la carte SD.
3. Insérez la carte SD dans la Switch.

5. Injection & Partition (TegraRcmGUI)

1. Sur PC, lancez TegraRcmGUI et installez les drivers (onglet Settings).
2. Branchez la Switch en RCM au PC.
3. Injectez le payload TegraExplorer.bin.
4. Sur la Switch : Allez dans "Partition the SD" -> "Fat32 + EmuMMC". Validez.
5. Une fois fini, redémarrez en RCM.

6. Créer l'EmuMMC & Lancer

1. Injectez le payload Hekate.
2. Menu Hekate -> emuMMC -> Create emuMMC -> SD Partition -> Part 1.
3. Une fois fini : Close -> Change emuMMC -> SD RAW 1.
4. Retour au menu -> Launch -> Atmosphere FSS0 EmuMMC.

IMPORTANT : Ne connectez jamais l'emuMMC à Internet (Nintendo) pour éviter le bannissement !

Informations
FAQ Switch (par Allan)

Qu’est-ce que ce tuto ?

Ce Tuto a pour finalité de prendre une Switch non modifiée à l'état "Usine" au CFW Atmosphère. Fusée-gelée est actuellement la meilleure méthode qui nous donne presque un contrôle total du système via le mode RCM.

Qu’est-ce que homebrew ?

Homebrew est un terme pour les logiciels non-officiels (éditeurs de sauvegardes, émulateurs, etc.). Ils peuvent être exécutés gratuitement via un CFW sur une Switch de "première génération" (non patchée).

Qu’est-ce qu'un Custom Firmware (CFW) ?

C'est un logiciel qui modifie le système (comme Atmosphère). Il tourne en arrière-plan et permet d'étendre les fonctionnalités, de modder des jeux via LayeredFS, et d'exécuter du code non signé.

Que puis-je faire avec un CFW ?

- Customiser le Menu Home (Thèmes).
- Utiliser des “ROM hacks”.
- Gérer ses sauvegardes (Checkpoint/JKSV).
- Jouer à des jeux rétros (RetroArch).
- Mettre à jour en sécurité.

Pourquoi une EmuNAND ?

L'emuNAND permet d'utiliser des homebrews et codes de triche sans affecter votre sysNAND (système original). Ainsi, votre sysNAND peut rester propre pour jouer en ligne sans risque de bannissement.

Qu’est-ce que le RCM ?

Le mode de récupération (Recovery Mode). Sur les Switchs non patchées, on peut envoyer un payload en RCM pour lancer Atmosphère.

Pourquoi ma Switch ne boot pas sur Hekate ?

Soit elle est Patchée (modèle trop récent), soit le Jig RCM est mal inséré.

Qu'est ce que l'emuNAND, l'emuMMC, sysNAND ?

sysNAND : Mémoire interne réelle de la console.
emuNAND : Copie complète de la NAND qui tourne sur la carte SD (séparée du système réel).
emuMMC : Nom technique de l'emuNAND utilisée par la Switch.

Message d'erreur Windows pour la carte SD ?

Si Windows demande de formater la carte SD, NE LE FAITES PAS ! C’est votre partition emuMMC que Windows ne reconnaît pas. Ignorez simplement le message.
